\begin{abstract}
Machine learning is commonly done in modern data science languages like Python, R, and Julia. Their ease of use and
simplicity makes them optimal for quickly developing neural networks, containerizing them, and serializing them for
sustained use. However, these languages are not chosen for maximum performance. Most modern languages selected
for machine learning (ML) are interpreted languages, typically relying on precompiled external libraries from lower lever
programming languages for vectorized calculations. Instead of relying on high-level interpreted languages or low-level
systems languages that risk undefined behavior, ML engineers can turn to modern, high-level systems programming
languages such as Rust to create safe and efficient neural networks.
\end{abstract}
